# 安全策略

## 受支持的版本

目前仅对最新版本提供安全更新支持。
请确保使用最新发布版本（可在 NuGet 获取）以获得安全修复与改进。

请始终使用最新版本 https://www.nuget.org/packages/MAS.Communication

## 报告漏洞

如果你发现潜在的安全问题，请通过以下方式报告：

- 提交 Issue（推荐标记为 security）
- 或联系维护者（如有私有沟通渠道优先）

请尽量提供：

- 问题描述
- 影响范围
- 复现方式（如适用）

## 回应政策

- 一般会在 1–3 个工作日内进行初步响应
- 确认问题后会尽快修复并发布更新
- 若判定为非安全问题，将在 Issue 中说明原因

## 说明

请不要在公开 Issue 中披露敏感漏洞细节（如未修复的利用方式）。
如涉及严重安全问题，建议先私下联系维护者。
